Intro to JavaScript

You may have a few questions about what JavaScript is, but I'm here to explain a few things!

But first off...

What is the purpose of JavaScript?

JavaScript was initially created to "make web pages come alive", it does this by giving programmers a variety of new abilities. JavaScript is able to add new HTML to a web page, modify existing content and even edit styles! JavaScript can also react to user input, like mouse clicks, key presses, and moving the mouse.

Who invented JavaScript and when?

JavaScript was developed by Netscape in 1995, and was originally called "LiveScript", but due to the popularity of Java at the time, it was changed to JavaScript likely as a marketing ploy.

JavaScript runs in the browser, of which I use Firefox.

Why are external scripts preferable to scripts added directly within an HTML file?

External scripts are preferred to scripts embedded into the HTML themselves for a few reasons, caching being the most important. Caching allows fewer bytes to be sent between the server and the client, which results in better performance. Modularity is another big plus as it allows you to use the same functions for multiple web pages.

What's the point of all those console.log() statements?

The console.log statements are used to evaluate the expression contained within the parenthesis. For example, "console.log(true + false)" logs "1" to the console, this is because boolean values are either 1 or 0, true or false. So when adding them, the console.log statement will log the result of 1 + 0.

What's the difference between how you reference a variable versus a string in a JavaScript statement?

To reference a string in a JavaScript statement, you include text within either single (') or double (") quotes. To reference a variable in a statement, you can use backticks to surround text, and then embrace the variable name in curly brackets ({}), preceded by a dollar sign ($).

Here's an example: 

                        
                            console.log("This is a piece of text"); // Writes 'This is a piece of text' to the console
                            // For the next example, we'll need a variable to use, so lets use 'x'.
                            let x = 1;
                            // This defines the variable 'x' to the integer value of 1.
                            console.log(`x is equal to ${x}`);
                            // This writes 'x is equal to 1' to the console
                            // If we change the value of x, this message would change as well!

What's the difference between a single value and an expression in a JavaScript statement?

A value contains one piece of information, such as an integer containing the value "1", or a boolean containing "true". An expression on the other hand, can contain multiple pieces of information.
