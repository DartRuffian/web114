Other Questions!

What is the document? (Choose from the following -- object, method, property)

The document is an object, and is the main "entry point" to the page.

What operator do you use to access the methods and properties of an object?

You can access the properties and methods of an object by the name of the object, followed by a period, and then the name of the method or object. Also remember that if you wish to call a method, you must include the parenthesis afterwards.


                            const person = {
                                first: "John";
                                last: "Smith";
                                full_name: function () { return this.first + " " + this.last; };
                            };
                            person.first;       // "John"
                            person.last;        // "Smith"
                            person.full_name(); // "John Smith"

What characters always follow a method?

Parenthesis must always follow a method. That includes when calling a method as well, which you can see in the example above.

What, in general, does a method of an object do?

Generally, a method does something. A method is an action that an object can perform. Imagine that an object contains a table of data, that object may have an add method, which would allow a user to add new data to the table.

What, in general, does a property of an object do?

While methods typically "do something", properties instead contain information. It may be the color of something, the name, an id, etc.
