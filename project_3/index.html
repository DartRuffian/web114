Identify the three major types of nodes found in an HTML document that we work with in JavaScript.

The three major types of nodes that we'll be working with in JavaScript are Element nodes, text nodes, and comment nodes.

Why is the JavaScript textContent property safer than innerHTML?

textContent is safer than innerHTML because it treats text as text. Say you were prompting a user for an input and they typed: <b>This is bold text</b>. If you used innerHTML, it would treat the bold tags as html, and (depending on the specific case), will either bold the text, or send only the text. However, using textContent would treat all parts of the text as text.

What do you notice about JavaScript properties and their corresponding HTML attribute names?

In the majority of cases, the JavaScript properties and HTML attribute names are the same, but there are a few cases where they are different.

Why do we rarely, if ever use document.write()?

One reason that document.write() is so rarely seen is because of it's age, as it was created before even the DOM and its standards were created. One flaw of this command is that it only works while the page is loading.

There are other issues with document.write(), which caused some browsers to drop support for it completely. Beginning with Chrome version 54, it no longer executes document.write() commands. One such issue is that it caused a large amount of delay or even prevented certain pages from loading completely due to poor network connection.

What is a CSS selector?

A CSS selector can be thought of as a pattern, that if an element fits, then it has that style applied to it. Some of these types of selectors are type selectors, class selectors, and ID selectors, but there are many many more.

What is a CSS property?

A CSS property is what allows developers to assign a style or behavior to a CSS selector. Examples include things like color, which changes the text color of an element.

What is a CSS declaration?

A CSS declaration consists of a CSS selector, at least one property, and then a value assigned to each of those properties.
