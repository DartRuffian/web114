Project 5: functions!! using both function declaration and function expression syntax

What is the difference between a function declaration and a function expression?

A function declaration creates a new function that can be later called by functionName(), while a function expression assigns a function to a variable, and is run as soon as it is defined.


                        // This is a function declaration
                        function hello() {
                            alert("HelloAlso");
                        }

                        // This is a function expression
                        var hello = function() {
                            alert("Hello");
                        }

Why do we create functions in the first place?

Functions are incredible tools no matter what programming language you're using. Functions allow us to prevent repeating code, two, three, or possibly even hundreds of times. After all, why copy/paste the same five lines of code, when you can put that in a function and call the function. Another reason that we use functions is to set when code is executed, such as listeners, event handlers, or whatever they may be called in the language you're using.
