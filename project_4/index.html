alert(), confirm(), and prompt() are methods of which object?

alert(), confirm(), and prompt() are all methods of the JavaScript window object.

Why can we use the alert(), confirm(), and prompt() methods without preceding them with the "object." syntax?

The reason we can use alert(), confirm(), and prompt() without having to specify the window part, is because it is a globally defined object. The window.method() syntax can still be used, but it is not required.

How do we distinguish variable names from strings in JavaScript?

In code, strings are surrounded by quotes, either single or double, while variables are not. Below is an example:


                        let this_is_a_variable = "This is a string!";

Strings represent text, whether it be letters, numbers, or other characters. While variables represent data.
